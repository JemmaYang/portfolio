Realtime Tweets
A Twitter Visualization of Climate Change
ROLE UI Designer & Programmer, Independent Work
DATE 11/2016 – 12/2016
TOOL Node.js, D3.js, JavaScript, HTML, and CSS3, Twitter Stream API
Design Statement

“Climate Change in Twitter” is a web application visualizing a real time discussion about climate change around world, based on geocoded data derived from tweets retrieved from a Twitter Streaming API. Each geocoded tweet is visualized as a circle located with a longitude and a latitude on the map. By mouse hovering the circle, you can view the content of the tweet. The application also displays top 10 hashtags mentioned by these tweets with a keyword “climate change” as a bar chart, which animates the number of each hashtag and their orders as the tweets updates, showing the most concerned issues within the topic of climate change. You can scroll down and review all tweets about the topic in a tweet board beside the bar chart.

Installation Instruction
  • Download and install Node.js from
    https://nodejs.org/
  • Download the project codes from terminal with a command:
    git clone https://github.com/JemmaYang/twitter-map
  • Set your own Twitter API key inside app.js file.
  • Install the dependencies inside the file directory with a command:
    npm install
  • Run the application with a command:
    node app.js
  • Type “localhost:8080” to your browser.
