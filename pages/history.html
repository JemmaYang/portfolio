TOOLHTML, CSS, JavaScript, A-frame.js, Argon.js
Design Goal

“Historical Tour” is a location-awareness project allowing users to explore Georgia tech’s campus and find old images, which were shot 50 years ago, of five campus buildings and obtain some information about these historical buildings. This project explores the possibilities, challenges and benefits of using responsive location data for interacting with users and creating a participative method of delivering educational messages. The project creates immersive experience for users through 3D scenes and the Argon browser simulating an augmented reality that corresponding with the location-awareness feature.
